Requirements
============

#. python http://python.org/download/
#. virtualenv https://pypi.python.org/pypi/virtualenv

Install
=======

Do this stuff::

    $ git clone https://github.com/willist/echochamber.git
    $ cd echochamber/
    $ virtualenv ve
    $ ve/bin/pip install -r requirements
    $ chmod u+x client.py master.py
    $ ./client.py
